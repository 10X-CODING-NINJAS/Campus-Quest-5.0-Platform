### Dimensional Anomaly Tracker

The Spider-Society needs to trace anomalies running across dimensions.

You are given an array of tracker signals and a target anomaly signature. Find **any pair of indices** whose signals sum to the target signature.

If multiple solutions exist, output **any** of them. If no solution exists, output `-1`.

#### Input Format
- First line contains two integers: $N$ (number of signals) and $Target$.
- Second line contains $N$ space-separated integers representing the signal values.

#### Output Format
- Two space-separated integers representing the indices, or `-1` if none exists.
