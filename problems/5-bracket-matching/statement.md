# Bracket Matching

Given a string containing only the characters `(`, `)`, `[`, `]`, `{`, and `}`, determine if the brackets are balanced.

Brackets are balanced if every opening bracket is closed by a corresponding closing bracket in the correct order.

## Input Format

* One line containing the bracket string.

## Output Format

* Print `VALID` if balanced, `INVALID` otherwise.

## Constraints

* `1 <= |s| <= 10^5`

## Sample Input 1

```
({[]})
```

## Sample Output 1

```
VALID
```

## Sample Input 2

```
([)]
```

## Sample Output 2

```
INVALID
```
