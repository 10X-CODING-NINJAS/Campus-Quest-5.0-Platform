### Dimensional Portal Sync

Synchronize two portals to construct a bridge back to Earth-1610.

Find any pair of key values from the portal array that sum exactly to the synchronization threshold.

Return the indices (0-indexed). If no pair matches, output `-1`.

#### Input Format
- First line contains two integers: $N$ and $Target$.
- Second line contains $N$ space-separated integers.

#### Output Format
- Two space-separated integers, or `-1`.
