### Glitch Signature Detector

Miles is glitching across dimensions! We need to calibrate his dimensional travel badge.

Find two glitch signatures in the data logs whose values add up to the stabilizer target.

Return the two indices (0-indexed).

#### Input Format
- First line contains two integers: $N$ and $Target$.
- Second line contains $N$ space-separated integers representing the glitch log values.

#### Output Format
- Two space-separated integers representing the indices.
